# Algorithm Visualizer

알고리즘의 작동 방식을 시각적으로 보여주는 웹 애플리케이션입니다. 학습용 도구로 활용하거나, 다양한 알고리즘의 동작 원리를 탐구하고 시각적으로 이해하는 데 도움이 됩니다.

## 주요 기능

- **다양한 알고리즘 시각화:**
  - **정렬 (Sorting):** 버블 정렬 (Bubble Sort), 선택 정렬 (Selection 정렬), 퀵 정렬 (Quick Sort) 등 주요 정렬 알고리즘의 단계별 과정을 시각적으로 보여줍니다.
  - **그래프 탐색 (Graph Traversal):** (구현 예정) 너비 우선 탐색 (BFS), 깊이 우선 탐색 (DFS) 등 그래프 탐색 알고리즘의 동작을 시각적으로 구현할 예정입니다. 현재는 개발 중에 있습니다.
- **다양한 시각화 뷰 지원:** 배열, 막대 그래프 등 여러 형태로 알고리즘의 실행 과정을 직관적으로 확인할 수 있습니다.
- **직관적인 사용자 인터페이스 (UI):** 시각화 패널, 상세 설명 패널, 로그 패널을 통해 알고리즘의 각 단계를 쉽게 추적하고 이해할 수 있도록 설계되었습니다.

## 기술 스택

이 프로젝트는 다음과 같은 기술 스택을 사용하여 개발되었습니다.

-   **Frontend:** React, TypeScript
-   **State Management:** Zustand
-   **Styling:** Tailwind CSS
-   **Build Tool:** Vite

## 시작하기

프로젝트를 로컬 환경에서 설정하고 실행하는 방법입니다.

1.  **저장소 복제:**
    프로젝트 저장소를 로컬에 복제합니다.
    ```bash
    git clone https://github.com/your-username/algorithm-visualizer.git
    cd algorithm-visualizer
    ```

2.  **의존성 설치:**
    프로젝트에 필요한 모든 의존성 패키지를 설치합니다.
    ```bash
    npm install
    ```

3.  **개발 서버 실행:**
    개발 서버를 시작하고 애플리케이션을 브라우저에서 확인합니다.
    ```bash
    npm run dev
    ```
    이 명령을 실행한 후, 웹 브라우저를 열어 `http://localhost:5173` (또는 콘솔에 표시되는 다른 포트)로 접속하면 애플리케이션을 확인할 수 있습니다.

## 현재 개발 상태 및 향후 계획

이 프로젝트는 활발하게 개발이 진행 중입니다. 현재는 몇 가지 정렬 알고리즘을 시각화하고 있으며, 앞으로 다음과 같은 기능들을 추가할 예정입니다.

-   더 다양한 정렬 및 탐색 알고리즘 (예: 힙 정렬, 병합 정렬, 다익스트라 등)
-   BFS, DFS 등 그래프 탐색 알고리즘의 실제 구현 및 시각화
-   사용자 정의 데이터 입력 및 시각화 옵션
-   성능 최적화 및 UI/UX 개선

여러분의 피드백과 기여는 프로젝트 발전에 큰 도움이 됩니다.
